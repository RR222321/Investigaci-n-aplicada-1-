Capítulo 1: Introducción a un Git
    - 1.1: ¿Qué es Git??
    -Es un sistema de control de versiones
    ampliamente utilizado en el desarrollo de
    software. Facilita el seguimiento de los
    cambios en los archivos y la colaboración
    entre múltiples personas en proyectos de programación.

    - 1.2: Instalación de Git
    -Descarga el instalador de Git para Windows haciendo clic en el enlace "Download for Windows".
     Ejecuta el archivo descargado para iniciar el instalador.
     Sigue las instrucciones del instalador, aceptando la configuración predeterminada en la mayoría de los casos. Puedes personalizar algunas opciones si lo deseas.
     Una vez completada la instalación, abre la línea de comandos ) y verifica la instalación escribiendo git --version. Deberías ver la versión de Git que has instalado
    
Capítulo 2: Flujo de trabajo básico
    -El flujo de trabajo básico en Git implica una serie de pasos que te permiten gestionar tus proyectos y colaborar con otros desarrolladores de manera efectiva.

Capítulo 3: Repositorios remotos
    -Los repositorios remotos en Git son versiones de tu proyecto alojadas en servidores en línea o en otros sistemas de control de versiones. 
    Estos repositorios te permiten colaborar con otros desarrolladores, mantener un respaldo seguro de tu código y compartir tu trabajo con el mundo.
